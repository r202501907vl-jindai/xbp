Arduinoを触ってみる

Arduinoとは？


入出力ポート（入力と出力ができる外部機器と信号のやり取りができる端子）がある
マイコンボード（電子機器を制御する板状の小型コンピュータ）のこと。
電子工作やプログラミングが簡単にできるプラットホーム。

Arduinoを使ってLチカ（LEDをチカチカさせる）


LEDのプラスマイナスの見分け方

長いほうが＋（アノード）
短いほうが－（カソード）という。

Lチカプログラム


要点

  1. Arduinoでは基本的に命令文の最後には;(セミコロン)をつける。
    ※｛｝でくくるループと条件分岐の場合は除く。

    2. デジタルピンの設定
  2. 「pinMode」:指定したピンを入出力どちらで使うか指定する。
  3. 「digitalWrite」:デザインピンの出力をLOW or HIGH（OFF or ON）で設定する。
  4. 「delay」:指定時間待たせる命令。単位はミリ秒(ms)

for文とif文

  1. for (初期化式; 条件式; 変化式){ // 繰り返しの中で実行される処理 }
  2. if (条件式){ 　処理内容1 　//条件式を満たす場合にのみ実行する }
  3. else{ 　処理内容2 　//条件式を満たさない場合にのみ実行される }

明るさセンサー

CdSセル：暗いところにおくと抵抗が強くなり、明るいところに置くと抵抗が弱くなる。
10kΩ抵抗：10,000Ωの電気抵抗を持つ電子部品。回路内の電流の流れを制限する。
